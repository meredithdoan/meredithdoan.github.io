MBUS

The Motorized Ballast Underwater System (MBUS) is a tethered, sensor-driven embedded vehicle designed to demonstrate real-time navigation, buoyancy control, and data telemetry. Developed as the final project for EECS 373: Introduction to Embedded System Design, the system combined integrated electronics, motor control, sensor processing, CAN communication, and a ground-station interface into a complete embedded platform.
